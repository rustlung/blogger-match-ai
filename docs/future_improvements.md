# Возможные направления развития

Этот список описывает реалистичные улучшения поверх текущей реализации.

## Источники и поиск

- добавить дополнительные поисковые провайдеры;
- поддержать другие социальные сети;
- использовать больше сигналов из открытых источников;
- сохранять историю найденных кандидатов между запусками.

## Скоринг

- добавить калибровку критериев под конкретные бренды;
- учитывать ручные оценки предыдущих запусков;
- вынести веса критериев в конфигурацию;
- добавить отдельные проверки качества аудитории при наличии данных.

## Операционная работа

- добавить CRM-статусы для кандидатов;
- фиксировать историю контактов;
- интегрировать email или мессенджеры после ручного подтверждения;
- хранить версии предложений и решений.

## Надёжность

- добавить retry-политику для отдельных внешних API;
- сделать отдельный режим запуска по сохранённым JSON;
- добавить отчёт по расходу внешних запросов;
- расширить диагностику частичных ошибок.

## Экспорт

- добавить более аккуратное форматирование Google Sheets;
- сохранять несколько запусков в отдельных листах или вкладках истории;
- добавить фильтры и условное форматирование для решений Matcher.
